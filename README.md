## 🚀 Sobre

O [Exercism](https://exercism.org/) é uma plataforma gratuita para aprender diversas linguagens de programação através de:

- 🧠 Exercícios práticos  
- 📖 Exemplos reais  
- 💬 Feedback da comunidade  

Cada vez que completo um exercício, ele é automaticamente versionado e enviado para este repositório.

---

## 🎯 Objetivo

- Praticar lógica de programação  
- Aprender múltiplas linguagens  
- Evoluir consistência em resolução de problemas  
- Manter um histórico público de aprendizado
